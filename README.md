# 🕊️ NayePankh Foundation Website

A modern, responsive NGO website redesign built for NayePankh Foundation. The project focuses on volunteer engagement, donation awareness, community impact, accessibility, and a premium user experience using HTML5, CSS3, and JavaScript.

## 🌟 Features

* Fully Responsive Design
* Modern NGO Landing Pages
* Volunteer Registration Form
* Donation Impact Section
* Success Stories & Testimonials
* Media Coverage Showcase
* Events & Campaigns Page
* Interactive Gallery
* Contact & Social Media Integration
* Dark / Light Mode Toggle
* Scroll Animations (AOS)
* Statistics Counter
* Gallery Filters
* FAQ Accordion
* Form Validation
* SEO-Friendly Structure
* Accessibility Improvements

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* AOS Animation Library
* Font Awesome
* Google Fonts
